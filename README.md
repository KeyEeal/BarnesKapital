# BarnesKapital

Production website for Barnes Kapital, built with React, TypeScript, and Vite.

## Scripts

- `npm run dev` - start the local Vite dev server.
- `npm run lint` - run ESLint.
- `npm run build` - generate the sitemap, type-check, and build the production bundle.
- `npm run preview` - preview the production build locally.
